# cs61-2018-material
This repository contains the pdf printup of all the lecture notes and section notes for the CS 61 course at Harvard in the Fall 2018 iteration. The notes come from here: https://cs61.seas.harvard.edu/site/2018/ and are publicly available
It also contains the section files (coming from here: https://github.com/cs61/cs61-sections) and lecture files (coming from here: https://github.com/cs61/cs61-lectures)
